Type
  • UITestServer

    This is the base for the UITest Local Server and it works in conjunction with UITestResponse It’s able to provide a bridge between UITest and the main application.

    UITestServer is able to mock Stateful and Stateless behaviors. By default it will be Stateless. Each response with a different endPoint pattern defines a new state chain if needed.
