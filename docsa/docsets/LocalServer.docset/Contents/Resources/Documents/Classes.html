  • StubResponse

    StubResponse is the base response to simulate URLResponses. It defines the HTTP Status code, the Responder Header and Body. It can also define a delay in seconds, simulating delayed server responses.

  • UITestResponse

    The UITestResponse is the base response for UITestServer. It contains methods to send the response data to the Local Server. This type also provides support to the concept of states for simulating Stateful behaviors

    • Basics: http://orca.st.usm.edu/~seyfarth/network_pgm/net-6-3-3.html
    • Concept: https://en.wikipedia.org/wiki/State_(computer_science)

    The response can be tied to a initial state and be connected to a next state, creating a state chain if needed.
